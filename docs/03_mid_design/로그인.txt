sequenceDiagram
    actor User
    participant Web_Server as 웹 서버
    participant Auth_Module as 사용자 인증 모듈
    participant Persistence_Module as 데이터베이스(등록된 사용자)

    User->>Web_Server: 1. 로그인 요청 (ID)
    Web_Server->>Auth_Module: 2. 사용자 인증 요청 (ID)
    Auth_Module->>Persistence_Module: 3. 사용자 정보 조회 (ID)

    alt 인증 성공 (ID 존재)
        Persistence_Module-->>Auth_Module: 4. 사용자 정보 (있음)
        Auth_Module->>Auth_Module: 5. 세션(Session) 생성
        Auth_Module-->>Web_Server: 6. 인증 성공 (Session_Token)
        Web_Server-->>User: 7. 로그인 성공 처리 (세션 설정)

    else 인증 실패 (ID 없음)
        Persistence_Module-->>Auth_Module: 4a. 사용자 정보 (없음)
        Auth_Module-->>Web_Server: 6a. 인증 실패
        Web_Server-->>User: 7a. 오류 메시지 표시

    end
