sequenceDiagram
    actor User
    participant Web_Server as 웹 서버
    participant Facade as 퍼사드
    participant Crawler as 크롤링&스크래핑
    participant AI_Analyzer as AI 분석

    User->>Web_Server: 1. 분석 요청 (URL, Keywords)

    alt 유효한 입력
        Web_Server->>Facade: 2. 리뷰 분석 요청 전달
        Facade->>Crawler: 3. 크롤링&스크래핑 요청 (URL)
        Crawler-->>Facade: 4. 크롤링된 결과 반환
        Facade->>AI_Analyzer: 5. 키워드 기반 분석/요약 요청 (크롤링 결과, Keywords)
        AI_Analyzer-->>Facade: 6. 분석/요약 결과 반환
        Facade-->>Web_Server: 7. 최종 분석 결과 반환
        Web_Server-->>User: 8. 결과 화면 출력 (분석 결과 ID 포함)
        note right of User: '저장' 버튼 출력

    else 유효하지 않은 입력
        Web_Server-->>User: 2a. 오류 메시지 반환 ("올바른 URL을 입력해주세요.")
    end
