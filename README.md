- 실질적으로 사람들의 관심을 끌 수 있는 지?

#### 낚시 및 해루질 관련 인지도 조사
- 문화체육관광부에서 발표한 자료에 따르면 2020년 코로나 팬데믹 이후 스포츠 낚시를 즐기는 인구가 증가
- 2021년 750만명, 2022년에는 1000만명을 돌파하여 스포츠 낚시에 대한 관심이 크게 증가하는 추세
- 트랜드에 따른 구독자 수 변화가 큰 유튜브에서 관련 유튜버들의 구독자 수가 크게 증가 (ex, 헌터팡, 진석기 시대, 힛뜨, 앵쩡 등)

### 프로그램을 개발해야한다고 생각한 이유
- 해안 경찰청의 보고에 따르면 낚시나 해루질을 즐기다가 물때를 확인하지 못해 발생하는 사고가 해년 증가.
- 해루질 동호회의 증가와, 관광객의 무분별한 해루질로 인해 어촌계의 큰 피해가 발생하고 있고, 그로 인한 갈등이 심화되고 있음
- 초보자가 물때를 정확하게 판단하기 어려울 뿐만 아니라, 현재 위치에서 육지까지 얼마나 걸리는 지에 대한 계산이 힘듦(육지에서 걷는 것과 속도가 다름)

### 구현하고자 하는 기능
- 낚시 및 해루질 장소 추천 기능 (지도 api, 어촌계의 해루질 제한 데이터)
- 날씨와 물때를 기반으로 날짜를 추천해주는 기능( 해양 조사원 물때 api, 기상청 api, 캘린더 api)
- 현재 각 지역의 금어기와 금지 체장, 제철 어류 등을 알려주는 지도 (해양수산부_수산정보_포획 금지 어종 정보,수산자원보호구역, 해양 수산부 제철 어류)
- +)원터치 구조신호 송출 기능, 시간과 위치 기반 위험 지역 경고 기능 (소방청, 해양 경찰청과의 협업이 필요)
