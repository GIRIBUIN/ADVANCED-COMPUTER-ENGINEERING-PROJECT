**프로젝트 아이디어** : 리뷰 분석 프로그램

**개요** : 별점이 높고, 추천 리뷰를 보고 구매하여도 불만족스러운 구매 경험을 하는 경우가 있음.

**프로그램 주요 기능 및 동작 방법**
- 해당 제품의 쇼핑몰 링크를 복사하여 붙여 넣음
- 해당 제품에 대해 중요하게 볼 관점을 사용자가 설정(키워드 입력 또는 클릭 등으로 선택)
- 분석을 시작하면 입력한 쇼핑몰의 링크에서 높고 낮은 평점의 리뷰를 각각 크롤링하여 입력받은 키워드를 중심으로 리뷰를 정리 및 출력

**발전** 
- 긍정적 리뷰와 부정적 리뷰에서 같은 내용의 리뷰 개수 표시(그 내용에 대한 신뢰도 평가를 위함)
- 최근 리뷰를 바탕으로 평점의 추세를 분석 및 표시(현재 시점에서의 평가 확인)
- 같은 품목의 제품들 중 이전에 분석된 제품들 중 더 괜찮은 제품이 있다면 추천(독자적 점수 체계를 만들어 링크와 함께 저장하는 식으로 구현)

**개발방법**
- 파이썬
- 웹
