# 최정길의 Branch

## 9/2 IDEA

### 수발주 매칭 프로그램 - 데이터를 구할 수 없어서 시연이 어려움

1. 메뉴(수주/발주)
  - 수주자(Vendor): 자신이 수주하는 제품을 필요로 하는 발주자를 매칭하는 것을 목적
  - 발주자(Client): 자신이 발주하는 제품을 제공하는 수주자를 매칭하는 것을 목적

2. 파일 연동(.xlsx, .csv, ...)
  - 수주자: 발주자의 정보를 관리 or 수주 제품 관리
  - 발주자: 수주자의 정보를 관리 or 발주 제품 관리 or 수주 업체 비교 정보 관리(가격, 제품, ...)

3. 개인화
  - 수주자: 수주 제품을 필요로 하는 발주 업체들 추천 리스트 제공
  - 발주자: 발주 제품을 제공하는 수주 업체들 추천 리스트 제공

4. API(확장)
  - 정부 지원 사업
    - 수주자: 자신의 업체에 맞는 지원 사업 제공
    - 발주자: 자신의 업체에 맞는 지원 사업 제공
   
5. 채팅
  - 매칭된 업체들끼리의 채팅 지원
  - 약속(meeting) 일정 관리 지원(캘린더 어플에 공유)

### 추가 고려 사항

1. 태그로 관리
  - 수/발주자 들의 업종 or 관심 제품 들을 태그로 관리하여 관심 업체끼리 매칭

2. 수/발주자 정보
  - 수주자: \[연락처, 위치, 사업자번호\], \[제품, 수주가능수량, 가격, 태그\]
  - 발주자: \[연락처, 위치, 사업자번호\], \[제품, 발주희망수량, 가격, 태그\]

3. 업종 카테고리 별 기본 가이드 제공
  - 커피와 철근 업종의 원하는 수발주 기본 정보가 다르니... 기본 가이드를 제공하는 방식 차용(태크 확장)

## 9/4 Idea

1. 청소 방법 알려주는 프로그램
2. 과일 보관방법, 신선도 일정 알려주는 프로그램
3. 벤치 지역 위치 알려주는 지도 프로그램
4. 영수증 분석 프로그램(공공데이터에서 평균 가격 따와서 내가 물건을 비싸게 샀는지 싸게 샀는지)
5. 한국에서 생산되는 제품을 외국어로 설명해주는 프로그램(포카칩의 상품명, 상품정보 등을 영어나 일본어, 중국어로 변환해주는 프로그램)

## 9/5 Idea

벤치 지도 프로그램
- 사용자의 기분에 따라 적합한 위치 추천해주는 서비스
    - 조용한 곳에서 책 읽고 싶어
    - 좋은 경치를 보고 싶어
    - 콘센트가 있는 벤치에서 노트북 작업 하고 싶어
- 데이터
    - 지자체 공원/녹지 서비스
    - 기상청 날씨, 미세먼지
    - 소음 정보
    - 주변 상권 정보

- 쇼핑 프로그램
   - 사용자의 특정 기간(이번 주는 한식!)
     - 일주일 동안의 식단을 제공
     - 해당 물품들의 가격(공공데이터) 제공(가능하면 마트 위치, 물품 가격)
     - 실제 내 구매 데이터와 비교 미래 지출 돕기

- 우리 동네 청정지수 프로그램
   - 쓰레기통 위치 지도(공공데이터)
     - +우리 지역 청정도 구분(녹->황->적)
   - 동네 쓰레기 치움 인증(봉사), 지역 커뮤니티
   - 민원서비스 포함하여 지차체와 소통까지 가능하도록
